# 🌠 UNIVERSE

## 📌 프로젝트 소개

- 전공별로 모이는 대학생 커뮤니티 플랫폼

## ✔ AI 기술 개요
  1. 학생증 OCR & 코사인 유사도를 이용하여 전공 간 유사도 추출 후 분류 
  2. 욕설/혐오 표현 탐지를 통하여 클린한 유저 환경 조성
  3. 미대생유저에게 해당 유저의 작품과 비슷한 스타일의 다른 유저의 작품을 추천
